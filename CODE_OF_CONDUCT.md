# Code of Conduct

We are committed to providing a friendly, safe, and welcoming environment
for all contributors.

## Expectations

- Be respectful and considerate.
- Assume good intent and communicate clearly.
- Focus on what is best for the community.
- Avoid harassment, discrimination, and personal attacks.

## Enforcement

Project maintainers may remove, edit, or reject contributions that do
not align with these guidelines.

If you experience unacceptable behavior, please contact the maintainers
via the security contact in `SECURITY.md`.
