# Contributing

Thanks for taking the time to contribute!

## Development setup

```bash
cd mail-forwarding-api/app
npm install
cp .env.example .env
npm run dev
```

## Tests

```bash
npm run test
```

## Lint

```bash
npm run lint
npm run format
```

## Pull requests

- Keep changes focused and small when possible.
- Include tests for bug fixes or behavior changes.
- Update documentation when you change configuration or behavior.

## Commit style

Use clear, imperative commit messages (e.g. "Fix token validation").
